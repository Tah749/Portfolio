# Profile Picture

Place your profile picture here as `profile-photo.jpg`

## Requirements:
- **File name**: `profile-photo.jpg`
- **Format**: JPG, PNG, or WebP
- **Size**: 400x400 pixels or larger (square format recommended)
- **Quality**: High resolution for best display

## How to add your photo:
1. Save your profile picture as `profile-photo.jpg`
2. Place it in this folder: `portfolio/public/images/profile/`
3. The image will automatically appear in your portfolio

## Alternative:
If you don't have a profile picture, the portfolio will show a placeholder image.
