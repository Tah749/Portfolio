# Factions Discord Bot Screenshots

Place your Discord bot project screenshots here.

## Required Files:
- `commands.jpg` - Bot commands and help system
- `faction-management.jpg` - Faction creation and management interface
- `leaderboard.jpg` - Player leaderboard and statistics tracking

## Requirements:
- **Format**: JPG, PNG, or WebP
- **Size**: 800x600 pixels or larger (16:9 aspect ratio recommended)
- **Quality**: High resolution screenshots
- **Content**: Clear Discord interface screenshots showing bot functionality

## How to add:
1. Take screenshots of your Discord bot in action
2. Save them with the exact filenames listed above
3. Place them in this folder
4. The gallery will automatically display them in your portfolio
