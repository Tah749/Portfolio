# FPL Visualization Project Screenshots

Place your FPL visualization project screenshots here.

## Required Files:
- `player-analysis.jpg` - Player performance analysis and statistics
- `team-stats.jpg` - Team statistics and performance metrics
- `visualization-dashboard.jpg` - Interactive dashboard with multiple visualizations
- `notebook-code.jpg` - Jupyter notebook with data analysis code

## Requirements:
- **Format**: JPG, PNG, or WebP
- **Size**: 800x600 pixels or larger (16:9 aspect ratio recommended)
- **Quality**: High resolution screenshots
- **Content**: Clear screenshots of data visualizations and analysis

## How to add:
1. Take screenshots of your FPL visualizations and Jupyter notebooks
2. Save them with the exact filenames listed above
3. Place them in this folder
4. The gallery will automatically display them in your portfolio
