# MovieFinder Project Screenshots

Place your MovieFinder project screenshots here.

- `homepage.jpg` - Homepage with featured movies and search functionality
- `search-results.jpg` - Search results with movie cards and filters

## Requirements:
- **Format**: JPG, PNG, or WebP
- **Size**: 800x600 pixels or larger (16:9 aspect ratio recommended)
- **Quality**: High resolution screenshots
- **Content**: Clear, well-lit screenshots showing key features

## How to add:
1. Take screenshots of your MovieFinder application
2. Save them with the exact filenames listed above
3. Place them in this folder
4. The gallery will automatically display them in your portfolio
